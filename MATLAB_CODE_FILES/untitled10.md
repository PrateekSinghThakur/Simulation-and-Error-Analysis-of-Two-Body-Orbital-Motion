# Two-Body Orbital Mechanics: Numerical Simulation & Analysis
## Comprehensive MATLAB Implementation - Modules 4 & 6

### PROJECT OVERVIEW

This project provides a complete implementation of numerical methods for solving the two-body gravitational problem in 2D. It demonstrates Module 4 (Interpolation) and Module 6 (Numerical ODE Solutions) through rigorous mathematical development, implementation from scratch, and comprehensive error analysis.
